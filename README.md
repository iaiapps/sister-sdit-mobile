## Features

- QR Code scan with GPS radius validation
- Manual attendance (sick, permission, official duty, early leave)
- Attendance history & recap
- Dynamic menu from server (Mutabaah, BPI, Substitute Teacher)
- Online/offline status indicator

## Tech Stack

Vue 3 | Vite 6 | Bootstrap 5 | Axios | Cordova (Android)
